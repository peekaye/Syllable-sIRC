User Accounts
Written by:Brent P. Newhall
Last Updated:Apr 11, 2007
Version:0.5.5 to 0.5.3

Syllable is a multi-user operating system, right from the ground up. When you first boot in to your Syllable installation, you'll find that you have to log in.

Syllable comes with two users pre-installed: root (username "root" and password "root") and guest (username "guest" and password "guest"). Root is the administrator user, and has complete access to the system. Guest is a regular user.

To create a new user, log in as root, double-click the Applications icon on the desktop, double-click the Users folder, and double-click the Users application.


Click the Add User... button to create a new user.

Let's say you're creating a user named Spike Spiegel. Enter the following information:

  • Name: Spike Spiegel
  • Login: sspiegel
  • Home: /home/sspiegel

Then click the OK button.

But you're not done yet! The Users application currently doesn't create the user's directory. You have to do that manually. Open a Terminal window and type the following:

$ cd /home
$ cp -a guest sspiegel
$ find sspiegel -print | xargs chown sspiegel:users

This creates Spike Spiegel's home directory, and makes everything in it owned by that user. Now the user is created, and you can log in as that user.
