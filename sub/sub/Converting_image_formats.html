Converting image formats
Written by:Nicolas
Last Updated:Apr 11, 2007
Version:0.3.4 to 0.5.3

You have to download the Image Magick package available on Kamidake. Then install it (very simple, see the INSTALL file). To convert a gif image to a jpeg one, you just have to type in the Terminal:

$ convert my_image.gif jpg:my_image.jpg

With Image Magick, you can convert one format to another, in any of 68 major formats. The syntax is as follows:

$ convert <image> <format> : <converted image>

For more information on Image Magick, www.imagemagick.org is a must-read (it's capable of blur, noise, mosaic effects, etc.).
