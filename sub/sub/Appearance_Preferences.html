Appearance Preferences

This preferences window lets you change the decor of Syllable to suit your own tastes. There are seven decors available:

  • Beish
  • Drow
  • Mulvane
  • Photon
  • Red
  • Win98
  • amiga

To change the decor, click the Decor drop-down, select a Decor, and click the Apply button.

You can also change the Themes within each Decor. Each Theme specifies a number of colors for each element of the Syllable user interface. The following elements are available:

  • Application – The background color for each application window
  • Highlight
  • Shadow
  • Active Window
  • Window Border
  • Menu Text
  • Menu Highlight Text
  • Menu Background
  • Menu Highlight
  • Scrollbar Background
  • Scrollbar Knob
  • Listview Tab
  • Listview Tab Text

To change the theme, select an element from the drop-down list, move the Red, Green, or Blue sliders or change the text fields, and click the Save Scheme button.

To delete a theme, select it in the Theme drop-down list and click the Delete Theme button.
