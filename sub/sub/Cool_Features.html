Cool Features

What does Syllable give you that can't find elsewhere?

First off, operating systems like Windows and MacOS have lots of legacy code to ensure that old software works properly. Syllable, on the other hand, isn't weighted down by the code needed to support decades' worth of old software. This makes Syllable responsive and lightweight.

Secondly, Syllable uses a cool file system called AFS, which can store lots of common data about files and lets you make searches on them like a database.

Thirdly, Syllable is POSIX-compliant. This means that Syllable can run a lot of tried-and-true Unix software, without needing tons of code to support it.

For developers, Syllable provides a clean, completely object-oriented programming API that's a dream to program in.

Syllable is currently being developed, but is stable and relatively mature. A number of useful Syllable applications are already available, and Syllable supports a wide range of common hardware device, including video, network, and sound cards from manufacturers such as Intel, AMD, 3Com, nVidia, and Creative.
