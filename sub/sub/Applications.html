Installing Applications

Syllable applications can be found at Kamidake.

Installing GUI/Window Applications

Start a Terminal window and untar the application. For example, if you're installing Juggler:

$ tar -xvpzf Juggler-1.0.0.tgz

That's it! You're done.

Installing Command-Line Applications

To install a command-line package such as emacs, start a Terminal window and do the following:

$ su -l
(Enter root's password)
$ cd /usr/
$ tar -xvpzf /syllable/packages/emacs-19.34.bin.tgz
$ pkgmanager -a /usr/emacs

This will make a bunch of symlinks in various subdirectories of /syllable/autolnk/ to make it possible for the system to find executables and other files belonging to the package.

Voila! You should now have the package installed.

Uninstalling Command-Line Applications

If you want to uninstall the package, you should clean out the symbolic links from /syllable/autolnk/ by running pkgmanager. For emacs:

$ pkgmanager -r /usr/emacs

Then delete the actual package directory itself. For emacs:

$ rm -R /usr/emacs
Please note that this way of installing applications should only be used for command-line applications and applications ported to Syllable from UNIX and other OS's using a Unix-like file system. Native applications should use the various mechanisms available in Syllable for making the application "position independent" so it can be installed anywhere on the hard drive. This mechanism includes the option of opening files using paths relative to the location of the executable files (and to obtain the path to the directory where the binary lives) and the ability to embed resources within the executable image itself.

Installing Non-Native Apps

How to install a collection of non-native apps (those that are supposed to go into /usr/) in just a few steps:

  1. Create a folder, say ~/packages
  2. Download all of the packages, saving them to ~/packages, without unzipping them.
  3. Type:
    4. $ cd /usr
    $ find ~/packages/ | xargs -n1 -i"{}" tar -xvpzf "{}"
    $ sync
    $ ls | xargs -n1 -i"{}" pkgmanager -a "{}"
    $ sync

Done!
